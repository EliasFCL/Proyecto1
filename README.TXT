=========================
COMO EJECUTAR EL PROGRAMA
=========================

1. Entrar a la carpeta dist
2. Hacer click derecho y abrir la terminal
3. En la terminal digitar lo siguiente: java -jar Proyecto1.jar calendario.pas y presionar enter
4. Luego del paso 3 aparecera un archivo de errores en la carpeta dist

Nota: Si la carpeta dist no existe se debe de ejecutar el proyecto en netBens, en la parte izquierda 
donde aparecen los proyectos se debe de hacer click derecho en el proyecto y seleccionar "Clean and Build"
luego de eso aparecera la carpeta dist
